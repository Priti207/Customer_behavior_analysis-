# Customer_behavior_analysis-
End-to-end data analytics project analyzing retail customer behavior and shopping trends using Python, SQL, and Power BI. Covers data cleaning, EDA, business analysis, and interactive dashboards, simulating real company-level workflows .

Overview

This end-to-end data analytics project analyzes retail customer behavior and shopping trends using Python, SQL, and Power BI. It demonstrates a real-world, company-level analytics workflow from raw data to insights and dashboards.

Dataset

Retail customer behavior and shopping trends dataset containing customer, product, and transaction details.

Tools & Technologies

Python – Data loading, cleaning, and EDA
SQL (PostgreSQL/MySQL/SQL Server) – Business analysis queries
Power BI – Interactive dashboard
Gamma – Presentation (PPT)
GitHub – Version control and documentation

Project Steps

Load and clean data using Python
Perform Exploratory Data Analysis (EDA)
Analyze business problems using SQL
Build an interactive Power BI dashboard
Create a project report and presentation

Dashboard

Power BI dashboard showcasing KPIs, customer segments, sales trends, and insights for decision-making.

Results

Identified key customer behavior patterns
Analyzed high-value customers and products
Generated actionable business insights

How to Run

Clone the repository
Run Python notebooks for data cleaning and EDA
Execute SQL scripts on your database
Open the Power BI file to view the dashboard.
