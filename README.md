# Customer_behavior_analysis-
End-to-end data analytics project analyzing retail customer behavior and shopping trends using Python, SQL, and Power BI. Covers data cleaning, EDA, business analysis, and interactive dashboards, simulating real company-level workflows .

Overview

This end-to-end data analytics project analyzes retail customer behavior and shopping trends using Python, SQL, and Power BI. It demonstrates a real-world, company-level analytics workflow from raw data to insights and dashboards.

Dataset

Retail customer behavior and shopping trends dataset containing customer, product, and transaction details.

Tools & Technologies

1.Python – Data loading, cleaning, and EDA
2.SQL (PostgreSQL/MySQL/SQL Server) – Business analysis queries
3.Power BI – Interactive dashboard
4.Gamma – Presentation (PPT)
5.GitHub – Version control and documentation

Project Steps

1.Load and clean data using Python
2.Perform Exploratory Data Analysis (EDA)
3.Analyze business problems using SQL
4.Build an interactive Power BI dashboard
5.Create a project report and presentation

Dashboard

Power BI dashboard showcasing KPIs, customer segments, sales trends, and insights for decision-making.

Results

1.Identified key customer behavior patterns
2.Analyzed high-value customers and products
3.Generated actionable business insights

How to Run

1.Clone the repository
2.Run Python notebooks for data cleaning and EDA
3.Execute SQL scripts on your database
4.Open the Power BI file to view the dashboard.
